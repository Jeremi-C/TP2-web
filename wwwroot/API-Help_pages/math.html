Aide pour l'API math

Endpoint: /api/maths

Paramètres:

  • op: L'opérateur (+, -, *, /, %, !, p, np)
  • x: premier paramètre
  • y: deuxième paramètre
  • n: seul paramètre

Réponse:

la réponse est objet JSON avec ces paramètres:

  • op: l'opérateur
  • x: premier paramètre
  • y: deuxième paramètre
  • value: le résultat de l'opération

Example:

Addition: /api/maths?op=+&x=3&y=2

Réponse:

{ "op": "+", "x": "3", "y": "2", "value": 5 }

Division: /api/maths?op=/&x=7&y=2

Réponse:

{ "op": "/", "x": "7", "y": "2", "value": 3.5 }

Modulo: /api/maths?op=%&x=7&y=2

Réponse:

{ "op": "%", "x": "7", "y": "2", "value": 1 }

Factoriel: /api/maths?op=!&n=-2

Réponse:

{ "op": "!", "n": "-2", "value": "Factorial must be positive" }

prime: /api/maths?op=p&n=7

Réponse:

{ "op": "p", "n": "7", "value": true }
